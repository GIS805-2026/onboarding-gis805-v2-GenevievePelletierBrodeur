# Dossiers de remise

Trois jalons de remise pendant le trimestre. Déposez vos artéfacts dans le
bon dossier **avant la date limite** indiquée ci-dessous.

| Dossier  | Jalon                          | Contenu attendu                                      | Échéance        |
|----------|--------------------------------|------------------------------------------------------|-----------------|
| `a1/`    | Revue de pairs 1 (après S04)  | Briefs S01–S04, SQL dims/facts jusqu'à S04, ai-usage | Fin de S04      |
| `a2/`    | Revue de pairs 2 (après S09)  | Briefs S06–S09, SQL dims/facts jusqu'à S09, ai-usage | Fin de S09      |
| `final/` | Remise finale (S12)           | Pack défense complet : briefs, SQL, docs/, ai-usage  | Date de S12     |

Les dates exactes sont annoncées sur Moodle et dans les guides de séance
(`docs/lab-guides/`). Consultez le tableau des livrables dans le
[`README.md`](../README.md) principal pour savoir quels artéfacts
correspondent à chaque séance.

## Comment déposer

1. Copiez (ou déplacez) vos fichiers dans le dossier du jalon concerné.
2. Committez et poussez sur `main` avant la date limite.
3. L'autograding CI vérifie automatiquement votre dépôt à chaque push.

> **Astuce :** les dossiers contiennent un `.gitkeep` pour que Git les
> suive même quand ils sont vides. Vous pouvez le supprimer une fois
> que vous y avez déposé vos fichiers.
